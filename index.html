BuscaCEP

Consulta de endereços via API própria (Node.js + Express)

Esse painel é a vitrine de uma API REST desenvolvida com Node.js e Express, que atua como uma ponte entre o cliente e o serviço público ViaCEP. A ideia é resolver um problema comum em sistemas web: receber um dado bruto do usuário, tratá-lo, buscar a informação em um serviço externo confiável, e devolver uma resposta limpa e estruturada — sem expor o cliente diretamente à API de terceiros.

Node.js Express Axios Docker ViaCEP

Como funciona

  1. Entrada: a API recebe o CEP dinamicamente pela URL (/cep/:cep)
  2. Sanitização: remove hífen e qualquer caractere que não seja número
  3. Validação: confere se sobraram exatamente 8 dígitos numéricos — se não, responde 400 na hora, sem gastar uma chamada externa à toa
  4. Consulta assíncrona: usa async/await com Axios para buscar o endereço no ViaCEP
  5. Resposta tratada: filtra os campos relevantes e devolve tudo em JSON — ou 500 se o serviço externo falhar
